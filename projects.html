Brain Inspired Computing Devices
Mimicking the nervous system and working of the brain to allow systems to know about their environment and modify their behavior based on this information is a long term goal of artificial neural networks. This project aims to build devices that display aspects of neuromorphic functions such as learning and plasticity. The problem statement is broadly divided into two categories.

  • Spike Firing Blocks: Blocks that replicate action potential of neurons. Different kinds of neurons exhibit different firing rates and are utilized in appropriate circumstances to model circuits. Regular Firing, Fast Firing, Burst Firing, Tonic Firing, Low Threshold Firing neurons are some of the models being analyzed. The goal is to utilize a single four terminal active element to replicate this behavior. Asynchronous Double Gate MOSFET with emphasis on Stress Induced Leakage Current traps and Frenkel-Poole defects through the oxide is being analyzed for this purpose.
  • Synaptic Blocks: Blocks that replicate synaptic behavior of the synapses are utilized here. Memristor models are utilized to model synaptic behavior. Linear ion Drift and Nonlinear Ion Drift models are analyzed. The goal is to emulate synaptic behavior effectiviely in minimal area and computational effort. The goal is to model a continuous chaotic oscillating two terminal device with a negative differential resistance region operating at the edge of chaos.
Software Used:   MATLAB, SMASH Simulator, SCROOGE Simulator, Yosys, QFlow, Graywolf, QRouter, NGSPICE, IRSIM.
Languages Used:   MATLAB, Verilog-AMS, Perl, Tcl, BASH.

Dynamic Neural Systems
This project aims to build Dynamic Neural Systems to work as different kinds of artificial neural networks. The main emphasis of the project is to realize Pulse Centered Neural Networks to be employed in various applications using the artificial neurons and synapses blocks being modeled. Artifical neural netoworks like simple feed forward systems, back propagation, classification, clustering and self organized feature maps are also being analyzed in addition to PCNN. The goal is to develop circuit specifications to be utilized with a crossbar array structure topology for synapses between neurons in the network.
Software Used:   MATLAB, SMASH Simulator, SCROOGE Simulator, Yosys, QFlow, Graywolf, QRouter, NGSPICE, IRSIM.
Languages Used:   MATLAB, Verilog-AMS, Perl, Tcl, BASH.

Mini Field Programmable Logic Array
The Mini-FPLA (Field Programmable Logic Array) consists of a matrix of R X C programmable logic units (PLUs). The objective of this project is to design a FPLA maximizing the number of PLUs and thereby maximizing the number of operations the FPLA can achieve. Each PLU contains an SRAM-based look-up table (LUT) and a flip-flop (FF). The vhdl code is used to test and analyze the functioning of the PLUs and the Mini-FPLA. The chip primarily has a mini-FPLA (Field Programmable Logic Array) which consists of a matrix of 4x6 programmable logic units (PLUs). Each PLU is designed for implementing a logic function. The PLU has to be programmed to implement any logic function either sequential or combinational logic. We have also included a stand-alone inverter, stand-alone inverter and a stand-alone D-flip flop. Features of FPLA: Operating voltage: 5V Number of pins: 38 Maximum Cock Frequency: 100MHz Max.
Software Used:   Modelsim, HSPICE, IRSIM, MAGIC.
Languages Used:   VHDL, SPICE, MAGIC Schematic.

Computer Aided Laser Incinerator
A Computer Aided Laser Incinerator is designed to etch patterns on a given surface in this project. The aim of this project is to build a bot whose motor functionality is controlled by an Arduino Uno chip which is inturn controlled by an FPGA chip (Altera Cyclone DE2 board) interfaced to a computer. Various IP cores are utilized to develop a system that maps keyboard input and projects the shape from the keyboard input on the VGA monitor. Once the input pattern is given, the FPGA converts the pattern into commands readable by the arduino uno chip on the bot which moves the bot based on the instructions to etch the given pattern using a laser on the surface.
Software Used:   Modelsim, Quartus, Arduino IDE, Notepad.
Languages Used:   Verilog, VHDL, C++, Python.

Decentralized Multi Robot Swarm Pattern Formation
A distributed multi-robot formation consisting of homogenous swarm of robots with simple and same abilities that can automatically search for patterns of simple shapes in a given environment is developed in this project. Generated robots cooperate locally to form interesting patterns. A swarm approach based on the Multi Object Particle Swarm Optimization model is used to generate randomness in the movement of each robot which is further aided by using an evaporating phermone technique to allow the robot to cover a given environment better. The group of robots follow a leaderless system with internal information sufficient for robot movement control. When a robot finds the pattern or shape provided in the environment, it secretes a special phermone on the edges of the pattern highlighting it and attracting other robots towards it.
Software Used:   MATLAB, Perl Intrepreter
Languages Used:   MATLAB, Perl.

Fuzzy Logic Kalman Filter on FPGA
The project proposes FPGA implementation of a fuzzy Kalman filter to be used for multiple purposes. The general idea is to use fuzzy logic and develop a Kalman filter algorithm to enhance the stability of estimated outcome respect to actual outcome with a high rate of error reduction and convergence of the system. Maximum Power Point Tracking Algorithms on FPGA
This project propses a novel approach to track MPP of a solar photovoltaic array using Kalman, Extended Kalman and Unscented Kalman Filter. Using either of these approaches, MPPT becomes much faster than using the conventional Perturb and Observe approach during sudden weather changes. Experiments were performed under optimal conditions as well as under cloudy conditions with falling irradiance levels. Comparative analysis resulted in Unscented Kalman Filter approach tracking MPP at a higher efficiency however Linear Kalman Filter approcah tracked MPP much faster. All the algorithms were implemented on a Cyclone 2 family FPGA device EP2C20F484C7.
Software Used:   Modelsim, Quartus Prime, MATLAB.
Languages Used:   VHDL, MATLAB, C.

PV Module Solaar Tracker System on FPGA
This project propses a novel approach to track MPP of a solar photovoltaic array using Kalman filter. Kalman Filter is also utilized to track the position of the PV array and change it to maximize the total output of the PV module for a simple and cost effective solution. By this approach, a motor to rotate the PV module and a piston to adjust the inclination of the PV module depending on irradiance levels to maximize output are realized to an error of +-2%. A melay type finite state machine with 5 states is used to control the system.
Software Used:   Modelsim, Quartus Prime, MATLAB.
Languages Used:   VHDL, MATLAB.
